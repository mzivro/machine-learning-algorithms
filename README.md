# Machine learning algorithms implementations

#### Exemplary miniprojects of implemented machine learning algorithms. I've written them for personal learning reasons and to reuse later if it'll be necessary. You can also use them, if you want.

### Supervised machine learning algorithms
* Linear regression
* Ridge and Lasso regression
* Polynomial regression
* Logistic regression
* k-nearest neighbors (KNN)
* Support Vector Machines (SVM)

### Datasets sources
* advertising.csv - https://www.kaggle.com/datasets/ashydv/advertising-dataset
* Position_Salaries.csv - https://www.kaggle.com/datasets/techwaker/position-salary-dataset
* score.csv - https://www.kaggle.com/datasets/himanshunakrani/student-study-hours
* Iris.csv - https://www.kaggle.com/datasets/uciml/iris
